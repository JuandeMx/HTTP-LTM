# Política de Privacidad de HTTP LTM

Última actualización: 21 de junio de 2026

En **HTTP LTM**, nos tomamos muy en serio tu privacidad. Esta Política de Privacidad describe cómo se recopila, utiliza y comparte la información cuando utilizas nuestra aplicación móvil.

## 1. Recopilación de Información de Navegación (VPN)
Nuestra aplicación funciona como un cliente de conexión segura (túnel SSH/VPN/Proxy).
* **Sin registros de tráfico (No-Logs Policy):** No recopilamos, registramos, almacenamos ni compartimos ningún registro de tu actividad de navegación, historial de búsqueda, destinos de tráfico, contenido de datos o direcciones IP de origen/destino mientras estás conectado a nuestra red VPN.

## 2. Información que Recopilan Terceros (Anuncios)
Para mantener la aplicación gratuita, integramos el SDK de **Google AdMob** para mostrar publicidad. Google AdMob puede recopilar y procesar cierta información de forma automática:
* Identificadores de publicidad del dispositivo (como el ID de publicidad de Android / GAID).
* Datos sobre cómo interactúas con los anuncios (clics, impresiones).
* Información técnica del dispositivo (modelo, versión de sistema operativo, proveedor de internet).

Puedes consultar la política de privacidad de Google para entender mejor el tratamiento de estos datos:
* [Política de privacidad de Servicios de Google](https://policies.google.com/privacy)

## 3. Seguridad
Implementamos medidas técnicas de seguridad (incluyendo ofuscación y cifrado) para proteger las configuraciones y transmisiones de datos dentro del app, evitando accesos no autorizados.

## 4. Cambios a esta Política
Nos reservamos el derecho de actualizar esta Política de Privacidad en cualquier momento. Te recomendamos revisarla periódicamente.
